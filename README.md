# 42lives

I am a working professional and AI-assisted builder focused on practical automation.

I started using AI because I wanted to automate repeated writing, documentation, repository, and workflow tasks. This GitHub profile is where I publish small local-first tools that make those automations safer, easier to review, and more reusable.

## Current Focus

- Maintainer automation for new open-source builders
- Repository preflight and release readiness
- Prompt workflow hygiene
- Reviewable LLM wiki/documentation workflows
- Privacy-conscious local-first tooling

## Projects

- [codex-maintainer-kit](https://github.com/42lives/codex-maintainer-kit) - local-first maintainer automation for AI-assisted open-source builders
- [promptops-toolkit](https://github.com/42lives/promptops-toolkit) - prompt linting and inventory tools for reusable AI workflows
- [llm-wiki-upgrader](https://github.com/42lives/llm-wiki-upgrader) - converts rough LLM notes into reviewable wiki pages

## Principles

- Keep sensitive checks local-first by default.
- Prefer reviewable workflows over one-off generated output.
- Document assumptions, risks, and release steps.
- Turn repeated personal automation into small public tools when it can help others.
